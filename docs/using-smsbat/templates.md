# Message Templates

Everything you need to know about creating and managing Viber/SMS templates.

## What is a Viber Template?
A Viber Service Message (Template) is a pre-defined message format that must be approved by Viber before sending.

## How long does approval take?
Typically, template moderation takes 1 to 3 business days.

## Can I edit a template after creation?
No, once a template is submitted for moderation, it cannot be edited. You must create a new one.
